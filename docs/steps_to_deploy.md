## How to deploy a angular static page to github

`npm install -g @angular/cli`

`ng new 21CenturyYogi`

` npm install --save d3`

`npm install -g angular-cli-ghpages`

`ng build --prod --base-href https://USERNAME.github.io/21CenturyYogi.git`
